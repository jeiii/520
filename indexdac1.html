开源跨平台的本地文件传输工具——LocalSend

默认分类 zhj 2023-09-01 08:27 318 0

在办公室上班的办公族经常需要整理和传输文件，有的时候还需要在手机以及 PC 或者不同设备间互相传输不同文件，大多数小伙伴习惯用的微信、QQ等聊天工具，但多多少少都会有一些限制，比如网络、大小、类型、隐秘性等，那么今天就分享一款好用的、支持全平台、完全开源的局域网文件互传工具——LocalSend。

软件介绍

LocalSend是一款免费、开源、跨平台的应用程序，使用REST API和HTTPS加密实现设备之间的安全通信，不需要连接互联网，就可让您通过本地网络与附近的设备安全地共享文件和消息。支持 Windows、macOS、Linux，以及 iOS、Android。

工作原理

LocalSend 使用自己的安全通信协议GitHub - localsend/protocol: The LocalSend REST API，允许设备使用 REST API 相互通信。所有数据都通过HTTPS安全发送，并在每台设备上动态生成TLS / SSL证书，确保最大的安全性。

下面，我们用Windows电脑和Android手机来演示。

Windows端

Windows版软件有绿色版本，无需安装，打开即可使用。

·金戈：电脑的别名

·#4：电脑的IP最末尾号

点击右上角的感叹号按钮，就可以看到详细信息

Android端

下载APK直接装到手机上即可。

当Windows端和Android端联通后，就能够在「发送」界面发现彼此。

在发送设备的「发送」界面，发送的内容可以以列表的形式发送多个文件，等整理好文件后，只需要选择需要接收的设备即可。

与此同时，在接收设备的「接收」界面，会有确认对话框提示接收，点击「接受」按钮才会开始传输文件。

此外，软件的设置也很简单方便，点击「设置」菜单，即可打开设置界面。像常用的「快速保存」、「保存目录」、「别名」等都可以设置。

今天的分享就到这咯，如果你也在寻找一款免费好用的文件互传工具，不妨下载LocalSend试试吧！
